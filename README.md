## GitHub Pages Notes

- Keep each task in its own folder: `Task-1`, `Task-2`, and so on.
- Each task folder should contain its own `index.html`, `style.css`, and assets.
- Use relative paths inside each task folder, such as `./assets/logo.jpg`.
- Do not use local drive paths in HTML/CSS.
- Folder names are case-sensitive on GitHub Pages, so link to `Task-1/` exactly.

## Local Preview
